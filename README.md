# To-Do List with File Storage (C++)

## Purpose
Command-line task tracker that writes all data to a local file. Direct add, list, mark-done, and delete operations.

## Features
- Plain-text storage (`tasks.txt`)  
- Append and rewrite modes  
- Auto-load on start  
- Indexed task operations  
- Minimal dependency footprint (STD only)
